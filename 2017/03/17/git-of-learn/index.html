git of learn

发表于 2017-03-17 | 
之前听说了github这个神奇的网站，自己也憧憬这五彩的learning世界中。

在没有人指导的情况下问了度娘! ，看了一些大神写的东西自己也搭建了一个Hexo!的Hello World。不过
很多东西都不知道怎么回事儿，这就导致了只能搁浅了github的学习。
有一天，

在一个偶然的机会下，我同学像强烈推荐github——。不过，我还是懵懂不知道怎么应用github和怎么去从小白到小白鼠。我就问啊问啊，我同学就像我推荐了
廖雪峰的git教程!从外面回来过后，我就开始一步步得跟着廖老师学习git。通过一个下午的实践get了：git的基本操作，blog base on next theme!
如果你也在迷茫中寻找git的学习，那么你也可以尝试一下。不一定能飞起来，但是你一定能get git or github!
